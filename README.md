# 📌 Overview

FocusGuard AI is an intelligent productivity and focus management platform designed to help students overcome distractions, improve concentration, and build effective study habits.

Modern students face constant interruptions from social media, entertainment platforms, notifications, and multitasking. These distractions significantly reduce productivity and negatively impact learning outcomes.

FocusGuard AI addresses this problem by combining focus tracking, distraction detection, productivity analytics, webcam monitoring, voice coaching, and goal management into a single web-based platform.

The system continuously monitors user behavior, provides real-time feedback, tracks study progress, and generates actionable productivity insights to help users achieve their academic goals.

---

Students often struggle to maintain focus during study sessions due to:

* Excessive social media usage
* Frequent tab switching
* Poor time management
* Lack of productivity awareness

* Absence of personalized feedback

These challenges result in:

* Reduced learning efficiency

* Stress and burnout

There is a need for an intelligent system that actively helps students stay focused and improve their productivity.

## 2. Focus Score System

The application assigns a dynamic Focus Score.

### Initial Score

100%

### Score Reduction

| Event                 | Penalty               |
| --------------------- | --------------------- |
| Tab Switch            | -10                   |
| Inactivity            | -5                    |
| Repeated Distractions | Additional deductions |

### Score Increase

| Event             | Reward |
| ----------------- | ------ |
| Completed Session | +5     |

### Purpose

The Focus Score serves as a measurable indicator of user concentration and study effectiveness.

---

## 3. Distraction Detection Engine

One of the most innovative features of FocusGuard AI.

### Tab Switching Detection

The system automatically detects when the user leaves the study page.

When detected:

* Distraction count increases
* Focus score decreases
* Alert notification appears
* Voice reminder is triggered

### Inactivity Monitoring

The application monitors:

* Mouse activity
* Keyboard activity

If no activity is detected for a predefined period:

* User receives a warning
* Focus score is updated

### Benefits

* Promotes accountability
* Encourages active participation
